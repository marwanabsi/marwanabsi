## 📦 Case Study: NGO Auto-Reporting Pipeline

**Problem:**  
> 20+ trainers submit activity data through chaotic Google Sheets. Reporting is manual, inconsistent, and error-prone.

**Solution:**  
✅ Automated ETL Pipeline  
- `Extract:` Google Sheets via Python API  
- `Transform:` Cleaned with Pandas (type corrections, fill missing, dedupe)  
- `Load:` Pushed to Azure SQL Database

✅ Dashboard Interface  
- Angular front-end to explore activities  
- Downloadable PDF reports  
- Role-based access

✅ Automation Layer  
- Prefect + Airflow combined scheduling  
- Email notifications for failures  
- Auto-deploy from GitHub via CI/CD

🎯 **Impact:**  
- Saved 80+ hours monthly  
- Improved data accuracy by 60%  
- Enabled real-time insights across 6 programs
